### git命令
- git init     //=>创建工具
- git add
- git add .
- git commit -m      //=>提交到暂存区
- git log        //=>查看历史版本记录
- git reflog     //=>查看所有历史版本
- git status     //=>查看状态
- git reset --hard+版本号      //=>回退


#### git和github
> GIT-HUB :https://www.github.com
一个网站(一个开源的源代码管理平台),用户注册后，可以在自己的账户下，创建仓库。用来管理项目的源代码(源代码基git创建到仓库中)
我们所熟知的插件、类库、框架等都在这个平台上有托管，我们可以下载观看和研究源码等

1.settings 用户设置
- profile 修改自己的基本信息
- Account Change username  修改用户名 个人页网址改变
- Security 可以修改密码
- Emails 修改邮箱
